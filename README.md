 Library Manager App

A simple, clean Library Manager frontend built with HTML, CSS, and JavaScript. This project helps practice DOM manipulation, event handling, and basic app structure — no frameworks, no magic, just solid fundamentals.

Features

* Add new books
* Display books dynamically
* Clean UI with basic styling
* Beginner-friendly structure



 Tech Stack

* **HTML5** – structure
* **CSS3** – styling
* **JavaScript (Vanilla)** – logic & interactivity



Project Structure


library/
 ├── index.html
 ├── style.css
 └── script.js


How to Run Locally

1. Clone the repository
2. Open the project folder
3. Open `library/index.html` in your browser

No server required.



Learning Goals

* Understand DOM manipulation
* Practice JS event handling
* Improve frontend project structuring
* Build confidence with real mini-projects


 Future Improvements
